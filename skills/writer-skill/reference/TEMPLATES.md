## 使用说明

### 模板使用原则
1. **灵活调整**：根据实际情况调整模板结构
2. **保留核心**：保持各阶段的核心要素
3. **补充细节**：在模板基础上补充具体内容
4. **删除无关**：删除不适用的部分

### 模板定制建议
1. 根据主题特点调整结构
2. 根据目标读者调整风格
3. 根据平台要求调整格式
4. 根据时间紧迫度调整详略

### 注意事项
1. 不要机械照搬模板
2. 重点是内容质量而非格式
3. 保持文档之间的一致性
4. 及时根据反馈调整模板
