# Writer Skill

Writer Skill 是一个系统化的文档写作助手工作流，帮助您将写作想法转变为高质量的文档内容。

## 概述

Writer Skill 采用六阶段工作流程，从需求分析到最终交付，确保每篇文档都经过深思熟虑，达到专业水准。

## 核心特性

- 📋 **系统化流程**：六个连续阶段，环环相扣
- 🎯 **质量把关**：多维度评分机制，客观评估内容质量
- 🤖 **AI味检测**：专门识别和改进AI生成内容的套路化表达
- 🔄 **迭代优化**：支持多轮优化，持续提升内容质量
- 📚 **丰富素材**：充分搜集网络资源，提供权威参考
- 🎨 **排版美化**：专业的格式优化，提升视觉呈现

## 六个阶段

### 1. 任务分析
深度理解写作需求，充分搜集资源素材，明确写作目标和受众。

### 2. 结构规划
设计文章标题、提纲目录和摘要，构建清晰的内容框架。

### 3. 内容创作
按照规划结构展开具体内容书写，注重实践导向和自然表达。

### 4. 质量复核
多维度评估文章质量，重点关注AI味问题，给出客观评分。

### 5. 优化改进
基于复核结果进行内容优化迭代，提升表达自然度和实用价值。

### 6. 排版美化
优化文章格式和视觉呈现效果，打造专业文档。

## 质量评分机制

Writer Skill 采用五维度评分体系：

| 维度 | 权重 | 说明 |
|------|------|------|
| 内容质量 | 25% | 信息准确性、逻辑完整性、价值密度 |
| 表达质量 | 20% | 语言流畅度、可读性、专业性 |
| 结构质量 | 20% | 层次清晰度、逻辑连贯性、重点突出度 |
| AI味评估 | 20% | 自然表达度、人性化程度、避免套路化 |
| 实用性评估 | 15% | 可操作性、实践指导价值、落地可行性 |

**综合得分标准：**
- ≥ 8.5分：优秀，可直接进入排版
- 6.0-8.4分：良好，建议优化改进
- < 6.0分：需改进，必须优化

## 使用方式

Writer Skill 会自动在相关场景下被 Claude 调用。您只需要：

1. **描述写作需求**：用自然语言告诉 Claude 您想写什么
2. **跟随工作流程**：Claude 会引导您完成六个阶段
3. **提供反馈**：在每个阶段确认或提出修改建议
4. **获得成果**：最终获得高质量的文档作品

## 核心约束

Writer Skill 遵循以下核心约束，确保工作质量：

- ✅ 必须严格遵循六阶段工作流，不能跳过
- ✅ 必须充分搜集网络资源和最新信息
- ✅ 每个阶段需要用户明确认可才能推进
- ✅ 支持反馈-修订循环，直到用户满意
- ✅ 使用 ultrathink 模式进行深度思考
- ✅ 保持从需求到交付的完整追溯链

## 特色功能

### AI味检测与改进

Writer Skill 特别关注 AI 生成内容的"AI味"问题，能够识别和改进：

- 套路化开头结尾（如"在当今这个...时代"）
- 机械化逻辑连接（如"首先...其次...最后"）
- 空洞化表达（如"显而易见"、"毋庸置疑"）
- 过度概括（如"众所周知"、"大家都知道"）
- 缺乏具体性（只有抽象概念，无具体案例）

### 优化循环控制

- 最多3轮优化循环
- 每轮优化后重新评分
- 自动终止条件，避免无限循环
- 记录优化轨迹和效果对比

### 自然语言交互

- 无需记忆命令，用自然语言描述即可
- 智能理解意图，自动导航到合适阶段
- 引导式确认，对话式推进
- 支持灵活调整和修改

## 最佳实践

1. **充分准备**：在任务分析阶段提供详细信息
2. **及时反馈**：每个阶段都认真review并提供反馈
3. **保持耐心**：好文章需要时间打磨
4. **善用优化**：不要满足于初稿，利用优化提升质量
5. **重视AI味**：特别关注自然表达度
6. **注重实用**：确保内容有实践指导价值

## 与斜杠命令的关系

Writer Skill 是对原 `/myWriter` 斜杠命令的升级重构：

- **相同点**：保留了完整的六阶段工作流程和所有约束
- **不同点**：
  - Skill 是**模型调用的**（Claude 自动识别使用场景）
  - 斜杠命令是**用户调用的**（需要显式输入 `/myWriter`）
  - Skill 的文档结构更加模块化和易于维护

您可以继续使用 `/myWriter` 命令，或者让 Claude 自动调用 Writer Skill。

## 常见问题

**Q: Writer Skill 会自动触发吗？**
A: 是的，当您的请求与文档写作相关时，Claude 会自动使用 Writer Skill。

**Q: 可以跳过某些阶段吗？**
A: 不建议。每个阶段都有其重要作用，跳过可能影响最终质量。

**Q: 优化阶段一定需要3轮吗？**
A: 不一定。如果综合得分达到8.5分，可以直接进入排版。最多3轮是为了控制时间成本。

**Q: 如何继续未完成的任务？**
A: 说"继续上次任务"（精准匹配该关键字），Claude 会检查 `.claude/writers/` 目录并继续。

**Q: 可以修改已完成的阶段吗？**
A: 可以。随时可以要求修改任何阶段的内容，系统支持灵活调整。
