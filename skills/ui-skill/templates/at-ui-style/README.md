# AT-UI风格模板

## 风格特点

- **主色调**：#E94609（橙红色）
- **设计风格**：现代、专业、动感
- **视觉效果**：渐变按钮、卡片投影、丰富的动画效果

## 组件特色

### 按钮
- 渐变背景（#E94609 → #D73A00）
- 悬停光泽动画
- 微妙的阴影效果

### 卡片
- 左侧色条装饰
- 悬停提升效果
- 渐变阴影

### 状态标签
- 渐变背景
- 圆点图标 + 脉冲动画
- 多种状态颜色

### 表格
- 悬停行高亮
- 微小提升效果
- 平滑过渡动画

### 模态框
- 毛玻璃背景
- 滑入动画
- 圆角设计

## 适用场景

- 通用B端管理系统
- 需要现代化视觉效果的项目
- 强调品牌色彩的系统
- 需要丰富交互反馈的应用

## 参考示例

- 采购订单管理系统
- 供应商管理系统
- 库存管理系统

## 技术特点

- 纯HTML + CSS + JavaScript
- 无需构建工具
- 直接在浏览器中运行
- 完整的CRUD交互
- 响应式设计

## 使用方法

在代码生成阶段，选择此模板，系统会基于template.html生成您需要的页面。
