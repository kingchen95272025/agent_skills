# Ant Design风格模板

## 风格特点

- **主色调**：#1890ff（蓝色）
- **设计风格**：专业、简洁、高效
- **视觉效果**：精简、一致、专业

## 适用场景

- 企业级应用
- 数据密集型系统
- 需要专业严谨形象的项目
- 大型团队协作的系统

## 模板文件

- `template.html` - 完整的Ant Design风格B端供应链采购订单管理页面模板

## 模板说明

此模板完整实现了采购订单管理的所有功能,包括:

### 核心功能
- 统计卡片展示
- 多条件筛选搜索
- 订单列表展示
- 完整的CRUD操作(新增、查看、编辑、删除)
- 批量操作(批量审批、批量删除)
- 数据导入导出
- 订单审批流程
- 表单验证

### 技术特点
- 单文件HTML,包含完整的CSS和JavaScript
- 使用模拟数据,无需后端支持
- 所有交互功能完全可用
- 响应式设计,支持多种屏幕尺寸
- 代码结构清晰,注释完善

### 使用方式
直接在浏览器中打开`template.html`即可查看和使用。

### 与其他模板的关系
- **HTML结构和JavaScript功能**与at-ui-style和element-plus-style模板**完全一致**
- **仅CSS样式不同**,采用Ant Design设计系统的视觉规范
- 可以无缝切换到其他风格的模板
