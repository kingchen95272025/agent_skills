# Element Plus风格模板

## 风格特点

- **主色调**：#409EFF（天蓝色）
- **设计风格**：清新、友好、易用
- **视觉效果**：柔和、舒适、亲和

## 适用场景

- 中后台系统
- 内部工具和管理系统
- 需要友好界面的应用
- 中小型团队使用的系统

## 模板文件

- `template.html` - 完整的Element Plus风格B端供应链采购订单管理页面模板

## 模板说明

此模板完整实现了采购订单管理的所有功能,包括:

### 核心功能
- 统计卡片展示
- 多条件筛选搜索
- 订单列表展示
- 完整的CRUD操作(新增、查看、编辑、删除)
- 批量操作(批量审批、批量删除)
- 数据导入导出
- 订单审批流程
- 表单验证

### 技术特点
- 单文件HTML,包含完整的CSS和JavaScript
- 使用模拟数据,无需后端支持
- 所有交互功能完全可用
- 响应式设计,支持多种屏幕尺寸
- 代码结构清晰,注释完善

### 使用方式
直接在浏览器中打开`template.html`即可查看和使用。

### 与其他模板的关系
- **HTML结构和JavaScript功能**与at-ui-style和ant-design-style模板**完全一致**
- **仅CSS样式不同**,采用Element Plus设计系统的视觉规范
- 可以无缝切换到其他风格的模板
