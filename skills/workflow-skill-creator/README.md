# Workflow Skill Creator

专业的Workflow Skill生成器，帮助您快速创建高质量的AI工作流Skill。

## 简介

Workflow Skill Creator基于Writer、PRD生成器、Kiro等成功案例的深度分析，掌握六阶段工作流的核心设计模式。通过系统化的四阶段流程，引导您从需求分析到最终交付，创建专业级的AI工作流Skill。

## 使用方式

这个Skill会被Claude自动调用，当您的请求匹配以下场景时：

- "我想创建一个workflow skill..."
- "帮我生成一个工作流提示词..."
- "需要设计一个AI工作流程..."
- "创建一个类似Writer/PRD/Kiro的skill..."

## 四阶段工作流程

### 第一阶段：需求分析与角色定位

- 通过系统化问卷收集基础信息
- 深度访谈了解行业特殊性和专业要求
- 生成完整的需求分析文档
- 明确AI助手的角色定位和核心价值主张

### 第二阶段：六阶段流程设计

- 设计符合需求的六个工作阶段框架
- 为每个阶段定义清晰的目标、输入、输出
- 设计专业的多维度评估体系
- 设计流程控制逻辑和阶段推进机制

### 第三阶段：Skill完整生成

- 基于通用模板进行系统化的变量替换
- 生成符合Agent Skill规范的SKILL.md主文件
- 生成六个阶段的详细实施指南（phases目录）
- 生成必要的参考文档（reference目录）

### 第四阶段：预览优化与交付

- 展示生成Skill的关键部分供预览
- 进行六维度专业质量检查
- 根据检查结果进行迭代优化
- 最终交付完整、高质量的Workflow Skill

## 质量标准

生成的Skill经过六个维度的专业检查：

1. **完整性** (20%) - 所有必需文件和章节齐全
2. **逻辑性** (25%) - 流程合理，阶段衔接顺畅
3. **专业性** (25%) - 领域知识准确，术语专业
4. **可用性** (15%) - 格式正确，可直接使用
5. **创新性** (10%) - 有独特设计亮点
6. **规范性** (5%) - 符合Agent Skill规范

综合得分≥8.0即可交付，≥9.0为卓越水平。

## 常见问题

### Q: 生成一个Skill需要多长时间？

A: 通常需要15-30分钟，取决于：
- 需求的复杂度
- 沟通的效率
- 定制的程度

### Q: 可以生成非六阶段的Skill吗？

A: 本工具专门用于生成六阶段工作流Skill。如果您的需求不适合六阶段结构，可能需要其他方式。

### Q: 生成的Skill可以修改吗？

A: 当然！生成的Skill是完全可编辑的。您可以：
- 直接编辑SKILL.md和其他文件
- 调整阶段设计
- 修改评估维度
- 优化交互方式

### Q: 如何让生成的Skill更专业？

A: 关键在于第一阶段的需求分析：
- 提供详细的领域信息
- 说明行业特殊性
- 提供实际案例
- 明确质量标准

### Q: 生成后如何测试？

A: 建议的测试方法：
1. 询问Claude："What skills are available?"
2. 提出匹配的请求，看是否触发
3. 完整走一遍工作流
4. 检查各阶段文件生成

### Q: 如何与团队分享？

A: 两种方式：
1. **项目Skill**：提交到 `.claude/skills/` 目录
2. **插件分发**：创建Claude Code插件
